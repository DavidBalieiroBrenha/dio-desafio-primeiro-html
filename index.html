    • Títulos HTML

    (Índice)

    Os cabeçalhos HTML, são definidos com as tags h1; h2; h3; h4; h5 e h6. h1 define o título mais importante e h6 define o título menos importante. Os mais utilizados são h1 para títulos e h2 para subtítulos.

    Parágrafos HTML

    (Índice)

    Os parágrafos HTML, são definidos com a tag p.

    Tags HTML mark; small; i; u e strong; sub; sup; e blockquote

    (Índice)

    A tag HTML mark, define o texto que deve ser marcado ou destacado.

    A tag HTML small, define um texto maior pequeno.

    A tag HTML i, define um texto em itálico.

    A tag HTML u, define um texto sublinhado.

    A tag HTML strong, define o texto em negrito.

    A tag HTML sub, define o texto subscrito, ou seja, aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor.

    A tag HTML sup, também define o texto subscrito, mais aqui é utilizado para escrever um texto com letras menores e com o alinhamento acima do resto do conteúdo.

    A tag HTML blockquote, é usada para fazer citações.

    Listas HTML

    (Índice)

    As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionais em listas. Uma lista não ordenada começa com a tag ul, enquanto a lista ordenada, começa com a tag ol, ou seja, os itens da lista serão marcados com números por padrão. Lembramos que cada item da lista começa com a tag li.

    Links HTML

    (Índice)

    Os links são encontrados em quase todas as páginas da web e permitem que os usuários cliquem de uma página para outra. Temos a tag HTML a, que vai definir um hiperlink. Seu atributo mais importante é o href, que indica o destino do link, ou seja, o texto do link é a parte que ficará visível para o leitor, que ao dar um clique no texto do link, enviará o leitor para o endereço de URL especifico.

    Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

    • Um link não visitado está sublinhado e azul
    • Um link visitado é sublinhado e roxo
    • Um link ativo está sublinhado e vermelho

    Temos também a tag hr, que é responsável pela quebra temática em uma página HTML (por exemplo, uma mudança de tópico). O hr é exibido com mais frequência como uma regra horizontal usada para separar o conteúdo ( ou definir uma alteração) em uma página HTML. É muito usado com o CSS, para alinhar um elemento, tirar sombra, definir altura e largura.
